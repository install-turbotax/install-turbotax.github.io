Could we install TurboTax on multiple devices?



For the online version of TurboTax, it’s accessible from any device with internet access since everything is cloud-based. You just need to log in to your account.

Important Considerations

  1. License Agreement: According to the InstallTurboTax.com End User License Agreement, you are allowed to install TurboTax on up to five computers that you own. Each installation must use the same user ID associated with your TurboTax account.
  2. License Code: You will need a valid license code for each installation. This code is typically found in your purchase confirmation email or on the product packaging if you bought a physical copy.
  3. Account Requirement: Starting from recent versions, TurboTax requires you to sign in to your Intuit account during the installation process. This links the software to your account and allows access to your tax return data across different devices.

Install TurboTax on multiple devices

Create or Sign In to Your TurboTax Account:

  1. Visit the i InstallTurboTax.com website: Sign in or create a new account if you don’t already have one.
  2. Purchase TurboTax: Ensure you have purchased the TurboTax software and keep your license code handy for installation.
  3. Install on the First Computer: Download the TurboTax installer from the official website or use a CD if applicable. Follow the installation prompts and sign in to your TurboTax account when prompted.
  4. Install on Additional Computers: On each additional computer, download the installer from your TurboTax account under the “Downloads” section. Install TurboTax following the same steps as before, ensuring you sign in with your account each time.
  5. Activate Each Installation: After installation, open TurboTax on each device and log in to your account. The software will recognize your account and sync your tax return data, allowing continuity across devices.

In summary, installing TurboTax on multiple devices is permissible under certain conditions, primarily limited to five installations per license code for personal use. Ensure that each device meets system requirements and that you have valid license codes for activation. By following these guidelines, you can efficiently manage your tax.
